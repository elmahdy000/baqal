# FULL CREATIVE LANDING PAGE PROMPT — Baqal / بقالك

## Project Name

**Baqal / بقالك**

## Page Type

Create a complete, premium, conversion-focused landing page for a SaaS + PWA grocery platform.

The page must be built as a serious SaaS startup landing page for Egypt and the Middle East, not as a generic food delivery website.

---

# 1. Product Summary

**Baqal / بقالك** is a SaaS platform that turns the local grocery store under residential buildings into an online ordering system.

The platform gives each grocery store a dashboard to manage:

- Products
- Product images
- Prices
- Stock quantities
- Orders
- Realtime notifications
- POS direct sales
- Customers
- Buildings
- QR codes
- Reports
- Settlements
- Subscriptions

Customers do not need to download a mobile app.

Residents scan a QR code placed inside:

- Building entrance
- Elevator
- Garage
- Reception
- WhatsApp group
- Printed flyer

The QR opens the store PWA directly.

The customer can:

- Browse products
- Add items to cart
- Enter apartment details
- Place an order
- Track order status
- Repeat previous orders

# 11. Design Rules

Apply these rules:

```txt

Use clean Arabic typography.
```
